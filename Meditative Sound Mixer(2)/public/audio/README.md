# Audio Files Setup

Place your meditation audio files in this directory following the structure below.

## Directory Structure

```
/public/audio/
├── chakra-1/
│   ├── track-1.mp3
│   ├── track-2.mp3
│   ├── track-3.mp3
│   ├── track-4.mp3
│   ├── track-5.mp3
│   ├── track-6.mp3
│   ├── track-7.mp3
│   ├── track-8.mp3
│   └── track-9.mp3
├── chakra-2/
│   ├── track-1.mp3
│   └── ... (tracks 2-9)
├── chakra-3/
│   └── ... (tracks 1-9)
├── chakra-4/
│   └── ... (tracks 1-9)
├── chakra-5/
│   └── ... (tracks 1-9)
├── chakra-6/
│   └── ... (tracks 1-9)
└── chakra-7/
    └── ... (tracks 1-9)
```

## Chakra Assignments

- **chakra-1**: Root Chakra (Muladhara) - 194.18Hz
- **chakra-2**: Sacral Chakra (Svadhisthana) - 210.42Hz
- **chakra-3**: Solar Plexus Chakra (Manipura) - 126.22Hz
- **chakra-4**: Heart Chakra (Anahata) - 136.10Hz
- **chakra-5**: Throat Chakra (Vishuddha) - 141.27Hz
- **chakra-6**: Third Eye Chakra (Ajna) - 221.22Hz
- **chakra-7**: Crown Chakra (Sahasrara) - 172.06Hz

## Track Suggestions

Each chakra should have 9 audio tracks. Here are the suggested track types:

1. **Track 1**: Ambient Pad
2. **Track 2**: Singing Bowl
3. **Track 3**: Nature Sounds
4. **Track 4**: Binaural Beat
5. **Track 5**: Chimes
6. **Track 6**: Drone
7. **Track 7**: Rain
8. **Track 8**: Wind
9. **Track 9**: Ocean Waves

## File Format

- **Format**: MP3 (recommended) or WAV
- **Sample Rate**: 44.1kHz or 48kHz
- **Bit Rate**: 128kbps or higher for MP3
- **Duration**: Recommended 2-5 minutes (tracks will loop automatically)

## Notes

- All tracks will loop continuously when playing
- Ensure all audio files are properly named (track-1.mp3 through track-9.mp3)
- The application uses the Web Audio API for playback and volume control
- Audio files are loaded when switching between chakras
